# 📈 Bitcoin Market Sentiment Analysis

## Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and cryptocurrency trading performance. It combines historical trading data with market sentiment to identify how investor emotions influence profitability.

---

## Objectives
- Analyze Bitcoin Fear & Greed Index data.
- Merge historical cryptocurrency trading data with sentiment data.
- Calculate trading performance metrics.
- Compare profit and loss across different market sentiment categories.
- Evaluate win rates under Fear, Neutral, and Greed market conditions.
